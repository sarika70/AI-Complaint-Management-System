# AI Complaint Management System

A web-based system that automatically classifies customer complaints using Machine Learning.

## Features
- Submit customer complaints
- AI categorizes complaints automatically
- Stores complaints in SQLite database
- Dashboard to view complaints

## Tech Stack
- Python
- Flask
- Machine Learning
- SQLite
- HTML
- CSS

## Project Structure
AI-Complaint-Management-System
│
├── app.py
├── model.py
├── complaints.db
├── templates
│   ├── index.html
│   ├── submit.html
│   └── dashboard.html
└── static
    └── style.css

## How to Run

1. Install Flask
pip install flask

2. Run the application
python app.py

3. Open browser
http://127.0.0.1:5000
